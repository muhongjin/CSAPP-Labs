# :rocket: CSAPP-Labs: A Programmer's Journey

> **"Computer Systems: A Programmer's Perspective"** 学习笔记与实验实现。

本项目记录了我深入底层系统、夯实 CS 基础的旅程。目标是完成 **CSAPP (3rd Edition)** 的所有章节阅读、课后练习以及 8 个核心实验室项目 (Labs)。

## :dart: 学习目标

* **透彻理解**：从二进制表示到异常控制流，打通软硬件隔阂。
* **实践至上**：独立完成所有 CMU 官方 Labs。
* **每日打卡**：保持 GitHub 提交绿点，记录思考过程。
